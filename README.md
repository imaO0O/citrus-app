# citrus-app
приложение для УП на тему "Сервис мониторинга ментального здоровья и антистресс-поддержки для студентов"
## Требования
- [Git]
- [Flutter](https://docs.flutter.dev/install/quick)
- [Docker Desktop](https://www.docker.com/products/docker-desktop)

## Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/imaO0O/citrus-app.git
   cd citrus-app
2. Запуск бд с помощью Docker:
   ```bash
   docker-compose up -d
3. Остановка работы с бд с помощью Docker:
   ```bash
   docker-compose down
