# Dart Frog Backend

Серверная часть приложения «Цитрус», реализованная на Dart Frog.

## Назначение
- Обработка REST API для всех сущностей (пользователи, дольки настроения, дневник, статьи, тесты, трекер сна и т.д.)
- WebSocket-сервер для чат-бота с ИИ-ассистентом (OpenAI)
- Аутентификация и авторизация (JWT)
- Интеграция с PostgreSQL (локальный Docker)
- Формирование аналитических отчётов (CSV/PDF)
