# Скрипты для смены IP-адреса API

## Быстрый старт

### Автоматически (рекомендуется)
Скрипт сам определит IP вашего компьютера:
```cmd
scripts\set-ip.bat
```

### Вручную
```cmd
scripts\set-ip.bat <IP> <PORT>
```

## Использование

### PowerShell
```powershell
# Автоматическое определение IP
.\set-ip.ps1

# Автоматическое определение IP с указанием порта
.\set-ip.ps1 -Port 8081

# Конкретный IP и порт
.\set-ip.ps1 -Ip 192.168.1.50 -Port 8081
```

### Batch-файл (Windows)
```cmd
# Автоматическое определение IP (порт 8081 по умолчанию)
set-ip.bat

# Только IP (порт 8081 по умолчанию)
set-ip.bat 192.168.0.100

# IP и порт
set-ip.bat 192.168.1.50 8081
```

## Примеры
```cmd
:: Локальный сервер
set-ip.bat localhost 8080

:: Сервер в локальной сети (автоматически)
set-ip.bat

:: Сервер в локальной сети (вручную)
set-ip.bat 192.168.0.100 8081

:: Другая сеть
set-ip.bat 10.0.0.5 3000
```

## Что меняется

Скрипт обновляет `baseUrl` в следующих файлах:
- `lib/core/repository/auth_repository.dart`
- `lib/core/api/sleep_api_service.dart`
- `lib/core/api/calendar_event_api_service.dart`

## Текущее значение по умолчанию
`http://192.168.0.101:8081`

## Для тестирования на Android-устройстве

1. Подключите телефон к **той же Wi-Fi сети**, что и компьютер
2. Запустите скрипт без параметров (автоматическое определение IP):
   ```cmd
   scripts\set-ip.bat
   ```
3. Включите отладку по USB на телефоне
4. Подключите телефон по USB
5. Запустите приложение:
   ```cmd
   flutter run
   ```

## Примечания

- Скрипт автоматически определяет Wi-Fi адаптер (исключая виртуальные адаптеры)
- Для корректной работы необходим PowerShell 3.0+
- Batch-файл использует UTF-8 кодировку (chcp 65001)
