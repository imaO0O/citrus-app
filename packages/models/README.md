# Shared Models

Пакет, содержащий общие Dart-модели данных, используемые как в бэкенде, так и во фронтенде.

## Назначение
- Единое описание структур данных (пользователь, запись дневника, отметка настроения и т.д.)
- Исключение дублирования кода
- Обеспечение согласованности типов между клиентом и сервером
