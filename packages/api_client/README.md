# API Client
Пакет, предоставляющий удобный интерфейс для взаимодействия Flutter-приложения с бэкендом.

## Назначение
- Инкапсуляция HTTP-запросов к REST API
- Управление WebSocket-соединением для чат-бота
- Обработка ошибок и токенов авторизации
- Предоставление единого сервиса для всех экранов приложения
